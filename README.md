# 🏠 EstateGPT – AI-Powered Real Estate Chatbot

EstateGPT is a full-stack AI-powered web application that enables users to search for properties using natural language queries. Instead of traditional filter-based systems, users can interact with a conversational interface to quickly find relevant property listings.

---

## 🚀 Features

* 💬 Conversational property search
* 🤖 Natural Language Processing (NLP) based query understanding
* 🏠 Property recommendation system
* ⚡ Real-time chat responses
* 📱 Responsive and user-friendly interface
* 🌐 Deployed web application

---

## 🧠 How It Works

1. User enters a query (e.g., “2BHK under 50 lakhs in Hyderabad”)
2. The system processes the query using NLP techniques
3. Extracts key parameters:

   * Location
   * Budget
   * Property type
4. Backend converts input into structured filters
5. Relevant properties are retrieved
6. Results are displayed in chat format
